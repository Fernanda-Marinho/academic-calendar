# Calendário Acadêmico UEFS

Este site reúne os calendários acadêmicos da UEFS em um só lugar e permite baixar os eventos para adicionar ao seu Google Agenda.

## Como usar

1. Escolha o calendário referente ao semestre que você deseja consultar.
2. Baixe o arquivo `.ics` gerado.
3. Abra o Google Agenda no computador e importe o arquivo.

Ao importar, você pode escolher em qual calendário os eventos serão adicionados. Para não misturar com seus compromissos pessoais, é recomendado criar um calendário separado, como **Calendário Acadêmico UEFS**.
Também é possível importar os eventos pelo celular, mas não há a possibilidade de escolher o calendário.
